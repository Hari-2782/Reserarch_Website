Real-Time
Road Safety Monitoring System

A comprehensive AI-Driven Multi-Agent System designed to monitor traffic rule violations, detect road hazards, analyze driver behavior, and perform risk analysis in real-time. Created for Sri Lankan road conditions without the need for expensive LiDAR sensors.

Explore the Domain

Domain Overview

Literature Survey & Research Gap

Current navigation systems depend mainly on manual hazard reporting, which is slow and unreliable. Furthermore, existing advanced driver-assistance systems (ADAS) rely on expensive sensors, like LiDAR, making them inaccessible for widespread application. Most existing vision-based systems focus solely on object detection and fail to ascertain exactly which situations require immediate driver attention. Furthermore, existing models are mostly trained on datasets representing right-hand traffic highways with clear markings, and simulators fail to perform well in real-world scenarios ("Sim-to-Real" gap).

Research Problem

How can we develop a highly reliable, low-cost AI-driven safety monitoring system that leverages real-time video feedback to accurately infer dynamic road hazards (potholes, rule violations, anomalies) without depending on GPS or expensive LiDAR sensors, specifically adapted to challenging environments such as Sri Lankan rural roads?

Research Objectives

  • • Develop a unified hazard detection (potholes, traffic signs).
  • • Conduct image-space risk analysis & anomaly detection.
  • • Detect road rule violations (lane tracking + speed estimation).
  • • Monitor driver alignment and simulate kinematics effectively on edge devices.

Proposed Methodology

  • • Pothole Validation: Applying Kalman filter temporal validation.
  • • Hazard Risk Reasoning: Evaluating YOLOv5, YOLOv7, and YOLOv8 architectures.
  • • Violation Detection: U-Net for lanes and Optical Flow for Speed.
  • • Behavior Simulation: Late-fusion hybrid architecture, trained partially via CARLA simulator.

1. Real-Time Pothole and Road Sign Detection

Dataset: Custom localized Sri Lankan road condition dataset alongside benchmark public datasets (e.g., RDD2020). This module demonstrates strong real-time performance on edge hardware. The multi-layer detection pipeline significantly improves accuracy by reducing geometric and classification errors.

Component Performance
YOLOv8 Detection Accuracy 82.0% mAP
Enhanced Detection (with filtering + OCR) 94.2%
Lane Detection (U-Net) 99.1% Accuracy
Speed Estimation ±10% Error
Processing Speed 15–30 FPS (Edge Device)

2. Risk Analysis, Road Hazard & Anomaly Detection

Dataset: Augmented custom hazard dataset covering sunny, rainy, and night conditions. This module uses physics-constrained filtering to eliminate spurious detections caused by sensor noise and environmental disturbances. It ensures only physically valid hazards are considered.

Metric Observation
Noise Robustness High
False Positive Reduction Significant
Anomaly Filtering Physics-based validation
Reliability Improved under noisy conditions

3. Road Rule Violation Detection

System Flow: A continuous real-time execution flow integrating sign detection, U-Net lane detection, and optical flow. The system detects violations such as overspeeding and lane deviation using integrated perception modules without GPS reliance. Events are logged with evidence for audit and analysis.

Metric Value
Detection Accuracy 94.2%
Violation Logging Real-time SQLite logging
Speed Compliance Detection ±10% accuracy
Lane Violation Detection 99.1% segmentation support

4. Driving Behavior & Alignment Monitoring

Dataset & Flow: Dual-source approach utilizing a 14-Hour digital twin simulation (CARLA) mapped to real-world IMU data. Hybrid architecture routes outputs via an Ensemble AI Filter and Kinematic Physics Layer. The model improves classification reliability and reduces hallucinations. The system achieves high precision while maintaining real-time edge inference capability.

Metric Value
Macro F1 Score 0.7224
Macro Precision 0.8165
Calibration Error (ECE) 0.041
Brier Score 0.024
Inference Latency 0.06 ms
External Validation Precision 93%

Key System Insights

  • Physics-guided fusion reduces false positives significantly
  • Soft fusion (α = 0.1) outperforms rigid rule-based systems
  • Edge deployment is feasible with low latency
  • System prioritizes precision over recall for safety-critical use
